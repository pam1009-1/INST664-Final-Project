# INST664-Final-Project
Code for INST664 final project analyzing language patterns in pet adoption profiles.

This repository contains the code for my INST664 final project, **"Finding Homes in Words: Analyzing Language Patterns in Pet Adoption Profiles."**

## Project Overview

This project analyzes pet adoption profile descriptions from the PetFinder.my Adoption Prediction dataset to examine whether linguistic patterns are associated with adoption speed.

The analysis includes:
- Data cleaning and filtering
- Description length analysis
- TF-IDF text classification
- Logistic Regression and Random Forest models
- K-Means clustering and PCA visualization
- Topic and theme comparison
- NRC emotion analysis
- VADER sentiment analysis
- Special-needs language analysis

## Main File

- `INST664_final_project_organized.ipynb`: Main notebook for the full analysis.

## Data

The project uses the PetFinder.my Adoption Prediction dataset from Kaggle.

## How to Run

Open the notebook in Google Colab or Jupyter Notebook and run the cells from top to bottom. The notebook assumes the PetFinder dataset files are available in the correct path or uploaded to the runtime environment.
